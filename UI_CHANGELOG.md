# 🎨 UI 更新说明

## 更新时间
2025-10-31

## 更新内容

本次更新根据 `UI界面/` 文件夹中的设计图，完全重新实现了前端页面，采用温暖的橙色系主题和清爽简洁的设计风格。

---

## 📱 客户端页面

### 1. 主页（菜品浏览）- `index.html`

**设计风格：**
- 🎨 主色调：橙色 `#FF8855` + 浅米色背景 `#FFF5F0`
- 🍜 Logo：圆角橙色图标 + "菜单"文字
- 🔍 搜索栏：圆角白色卡片，带搜索图标

**功能特性：**
- 登录弹窗：点击"登录"弹出居中模态框
- 分类标签：横向滚动，橙色高亮选中状态
- 菜品卡片：
  - 左侧图片占位符（渐变橙色）
  - 右侧信息：菜名、描述、价格、库存
  - 圆形加号按钮（橙色）
  - 热销标签（浅橙色背景）
- 悬浮购物车：右下角圆形按钮，红色数量徽章

**响应式适配：**
- 电脑端：多列网格布局
- 手机端：单列布局，图片缩小至80px

### 2. 购物车/结算页 - `checkout.html`

**页面结构：**
- 顶部栏：返回按钮 + "购物车" 标题 + 商品数量
- 支付提示：黄色警告框，提示线下付款
- 订单列表：白色卡片，显示菜品名、规格、数量、小计
- 底部固定栏：合计金额 + 橙色提交按钮

**订单成功页：**
- 绿色圆形对勾图标
- 订单号：橙色大字号显示（格式：#A2025103001）
- 订单详情卡片：下单时间、数量、金额
- 返回首页按钮：橙色圆角
- 查看详情链接：橙色文字 + 箭头

## 🖥️ 商家后台页面

### 1. 登录页 - `admin.html`

**设计风格：**
- 居中白色卡片（圆角20px）
- 橙色机器人图标（80px圆角）
- 标题：橙色 "商家管理后台"
- 副标题：灰色 "点单程序网 Web 管理系统"

**表单元素：**
- 输入框：圆角8px，灰色边框
- 登录按钮：全宽橙色，圆角10px
- 返回链接：橙色文字

### 2. 数据仪表盘

**布局：**
- 左侧边栏：导航菜单（200px宽）
  - 📊 数据概览（高亮橙色）
  - 📋 订单管理
  - 🍽️ 菜品管理
  - 📦 库存管理

**数据卡片（4个）：**
1. 今日订单数：128（绿色趋势 ↑ +12%）
2. 今日营业额：¥3,482（橙色金额）
3. 待处理订单：12（橙色警告）
4. 已完成订单：116（绿色完成率）

**最新订单表格：**
- 列：订单号、时间、数量、金额、状态、操作
- 状态标签：蓝色（已提交）/ 绿色（已完成）/ 红色（已取消）
- 操作按钮：绿色"完成"按钮

### 3. 订单管理

**筛选标签：**
- 全部订单（高亮橙色）
- 待确认 / 已完成 / 已取消（白色圆角）

**表格字段：**
- 订单号、时间、菜品明细、数量、金额、状态、操作
- 详情按钮（蓝色）+ 完成按钮（绿色）

### 4. 菜品管理

**网格布局：**
- 3-4列自适应网格
- 每张卡片：
  - 图片占位符（160px高，渐变橙色）
  - 菜品名（粗体）
  - 信息：分类 · 库存 · 状态
  - 价格：橙色大字号
  - 三个按钮：在售（绿）/ 编辑（橙）/ 下架（红）

### 5. 库存管理

**控制卡片（2个）：**
1. 调整库存：输入菜品ID + 调整量（+/-）
2. 上下架菜品：输入菜品ID + 选择状态

**库存预警表格：**
- 仅显示库存 < 20 的菜品
- 红色/橙色字体标注库存紧张
- 快速补货按钮（绿色）

## 🎨 设计系统

### 配色方案

```css
/* 主色 */
--primary-orange: #FF8855;
--primary-hover: #FF7744;

/* 背景 */
--bg-light: #FFF5F0;
--bg-white: #FFFFFF;
--bg-gray: #F8F9FA;

/* 文本 */
--text-dark: #333333;
--text-gray: #666666;
--text-light: #999999;

/* 边框 */
--border-light: #E5E5E5;
--border-gray: #F0F0F0;

/* 状态 */
--success: #4CAF50;
--warning: #FF9800;
--error: #F44336;
--info: #2196F3;
```

### 圆角规范

- 卡片：12px
- 按钮：8-10px
- 标签：20px（圆角胶囊）
- Logo/头像：20px

### 阴影效果

```css
/* 卡片阴影 */
box-shadow: 0 2px 8px rgba(0,0,0,0.05);

/* 悬浮阴影 */
box-shadow: 0 4px 12px rgba(255, 136, 85, 0.4);
```

## 📱 响应式断点

```css
/* 移动端 */
@media (max-width: 768px) {
  - 侧边栏收缩至60px（仅显示图标）
  - 菜品卡片单列布局
  - 数据卡片单列堆叠
  - 表格横向滚动
}
```

## ✨ 新增交互特性

1. **登录模态框**：点击背景关闭
2. **悬浮效果**：卡片/按钮 hover 时轻微上移
3. **加载动画**：数据请求时的加载状态
4. **实时反馈**：库存不足时按钮禁用，颜色变灰
5. **平滑过渡**：所有交互均添加 0.3s 过渡动画
